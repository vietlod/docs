# Hướng dẫn nhanh (Quick Start)

Quy trình 6 bước để đi từ "mở ứng dụng" đến "có bộ dữ liệu + phân tích". Mỗi bước có liên kết tới tài liệu chi tiết. Bản tương tác đầy đủ nằm trong ứng dụng tại [ecodata.io.vn/quick-start](https://ecodata.io.vn/quick-start).

## 1. Đăng nhập

Đăng nhập bằng email hoặc Google. Sau khi vào, bạn ở **Dashboard** — nơi tìm kiếm indicators quốc tế và truy cập các module dữ liệu Việt Nam. Duyệt danh mục và xem trước là miễn phí; xuất đầy đủ cần gói thuê bao.

→ Chi tiết: [Login → Filter → Preview → Export](/ecodata/bat-dau/login-filter-preview-export)

## 2. Khám phá nguồn dữ liệu

Chọn nhóm theo **đơn vị phân tích** của câu hỏi nghiên cứu:

| Bạn cần | Vào module | Tài liệu |
| --- | --- | --- |
| So sánh quốc tế, biến kiểm soát vĩ mô | Global Data | [Global Data](/ecodata/indicators/global-data) |
| Số liệu chính thức Việt Nam theo tỉnh/năm | GSO Explorer | [GSO Việt Nam](/ecodata/indicators/gso-vietnam) |
| Xuất nhập khẩu theo mặt hàng/đối tác | Customs Explorer | [Customs](/ecodata/indicators/customs) |
| Quản trị địa phương (PCI/PAPI/...) | Survey Explorer | [Macro Survey & VHLSS](/ecodata/indicators/macro-survey-vhlss) |
| Vi mô hộ gia đình | VHLSS Variable Hub | [Macro Survey & VHLSS](/ecodata/indicators/macro-survey-vhlss) |
| Giá, BCTC, sự kiện doanh nghiệp | Stock Hub | [Stock Hub](/ecodata/indicators/stock-hub) |

→ Tổng quan: [Phân nhóm nguồn dữ liệu](/ecodata/nguon-du-lieu/phan-nhom-nguon-du-lieu)

## 3. Chọn & lọc indicators

1. Nhập **từ khóa** hoặc **short code** (vd `wb_mac_gdp_usd`).
2. Lọc theo **nguồn** nếu muốn giới hạn vào một tổ chức.
3. Chọn **tần suất** phù hợp với mô hình (năm/quý/tháng).
4. Đặt **khoảng năm** để loại chỉ tiêu không đủ giai đoạn.
5. Ưu tiên chỉ tiêu có **completeness** cao.

*Ví dụ đầu vào:* `Tỉnh = TP.HCM, chỉ tiêu = GRDP, 2015–2023`.

→ Khái niệm trường dữ liệu: [Từ điển dữ liệu](/ecodata/du-lieu/data-dictionary)

## 4. Xem trước

Trước khi tải toàn bộ, hãy kiểm tra ba điểm: **coverage** (đủ quốc gia/tỉnh và đủ năm?), **đơn vị** (cùng đơn vị khi ghép nguồn?), và **định nghĩa nguồn** (cùng tên nhưng khác cách tính?).

→ Vì sao quan trọng: [Phương pháp luận](/ecodata/du-lieu/methodology)

## 5. Xuất dữ liệu

Xuất ra **CSV, Excel (XLSX) hoặc JSON**, kèm **metadata + short code**. Số lượt xuất/ngày tùy theo gói.

*Ví dụ đầu ra:* bảng panel `country/province × year × indicator` + tệp metadata mô tả đơn vị, tần suất, nguồn.

→ Định dạng & hạn mức: [Preview & Export](/ecodata/xuat-du-lieu/preview-export)

## 6. Phân tích kinh tế lượng

Dùng công cụ **Kinh tế lượng** tích hợp để chuẩn bị dữ liệu, ước lượng và kiểm định mô hình (OLS, WLS, GLS, panel, VAR, ARIMA, Logit, Probit…), trình bày bảng kết quả theo chuẩn trích dẫn, rồi **xuất bộ mã đầy đủ chạy trên Stata/R/Python** để tái lập.

→ Chi tiết: [Kinh tế lượng](/ecodata/cong-cu/econometrics) · Trợ lý: [AI Chat](/ecodata/cong-cu/ai-chat)

## Truy xuất dữ liệu bằng Excel / Python / Stata / R

Nếu workflow nghiên cứu của bạn chạy trong Excel, Python, Stata hoặc R, hãy dùng **Bộ công cụ truy xuất dữ liệu EcoData** thay vì tải thủ công rồi sao chép bảng. Tất cả client dùng cùng Public SDMX API `/api/v1/sdmx` và cần khóa API cá nhân.

### 1. Tạo khóa API

1. Vào **Dashboard → Truy cập API**.
2. Tạo khóa mới, đặt tên theo dự án hoặc phần mềm sử dụng.
3. Sao chép khóa đầy đủ một lần và lưu an toàn trong biến môi trường, `~/.ecodata`, Office task pane storage hoặc trình quản lý mật khẩu.
4. Thu hồi khóa khi dự án kết thúc hoặc thiết bị/cộng tác viên không còn cần truy cập.

### 2. Chọn dataflow

| Dataflow | Khi nào dùng |
| --- | --- |
| `MACRO_INTL` | Chỉ số vĩ mô quốc tế theo indicator, quốc gia và tần suất. |
| `GSO_VN` | Chỉ số GSO Việt Nam theo indicator, tỉnh/thành, dataset và tần suất. |
| `CUSTOMS_VN` | Panel xuất nhập khẩu theo tỉnh/thành, luồng, thước đo và tần suất. |
| `STOCK_VN` | Chuỗi thị trường Việt Nam theo mã, metric và tần suất. |

### 3. Chạy trong phần mềm nghiên cứu

```python
from ecodata import EcoData

eco = EcoData()
gdp = eco.series(
    "MACRO_INTL",
    indicator="NY.GDP.MKTP.CD",
    ref_area=["VNM", "USA"],
    freq="A",
    start=2010,
)
```

```r
library(ecodata)

gdp <- eco_series(
  "MACRO_INTL",
  indicator = "NY.GDP.MKTP.CD",
  ref_area = c("VNM", "USA"),
  freq = "A",
  start = 2010
)
```

```stata
net install ecodata, from("https://ecodata.io.vn/clients/stata") replace
ecodata setkey eco_live_...
ecodata use MACRO_INTL, indicator(NY.GDP.MKTP.CD) ref_area(VNM USA) freq(A) start(2015) frame(gdp)
```

```text
=ECODATA.SERIES("MACRO_INTL","NY.GDP.MKTP.CD.VNM.A","2019","2023")
```

**Trạng thái phát hành:** Python, R và Stata core client đã pass live smoke production với dữ liệu thật; Stata và Excel đã có route self-host. PyPI, CRAN, SSC, AppSource và kiểm thử runtime Excel thủ công vẫn là các mốc phát hành riêng, chưa nên mô tả như đã hoàn tất.
