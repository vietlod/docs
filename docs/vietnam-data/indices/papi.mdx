# Provincial Public Administration Performance Index (PAPI)

**PAPI** measures citizens' experiences with public administration and governance.

## About PAPI

- **Publishers:** UNDP, CECODES, VFF-CRT
- **Started:** 2009
- **Coverage:** 63 provinces
- **Methodology:** Citizen survey (16,000+ respondents)

## PAPI Dimensions (8)

| Dimension | Description |
|-----------|-------------|
| Participation | Civic engagement |
| Transparency | Government openness |
| Vertical Accountability | Responsiveness to citizens |
| Control of Corruption | Anti-corruption efforts |
| Public Administrative Procedures | Service delivery efficiency |
| Public Service Delivery | Quality of public services |
| Environmental Governance | Environmental management |
| E-Governance | Digital government services |
