# Vietnam ICT Development Index

The **ICT Index** measures information and communication technology development.

## About ICT Index

- **Publisher:** Ministry of Information & Communications
- **Frequency:** Annual
- **Coverage:** 63 provinces
- **Components:** Infrastructure, Human capital, Applications

## ICT Index Components

| Component | Weight | Indicators |
|-----------|--------|------------|
| ICT Infrastructure | 35% | Internet, mobile coverage |
| Human Capital | 30% | IT skills, education |
| ICT Applications | 35% | E-government, e-commerce |
