# Vietnam PMI - Purchasing Managers Index

The **Vietnam Manufacturing PMI** is a leading indicator of manufacturing sector health.

## About PMI

- **Publisher:** S&P Global
- **Frequency:** Monthly
- **Coverage:** Manufacturing sector
- **Scale:** 0-100 (50 = neutral)

## PMI Interpretation

| Score | Meaning |
|-------|---------|
| > 50 | Expansion |
| = 50 | No change |
| < 50 | Contraction |

## PMI Components

| Component | Weight | Description |
|-----------|--------|-------------|
| New Orders | 30% | Demand indicator |
| Output | 25% | Production levels |
| Employment | 20% | Workforce changes |
| Suppliers' Delivery Times | 15% | Supply chain |
| Stocks of Purchases | 10% | Inventory |
