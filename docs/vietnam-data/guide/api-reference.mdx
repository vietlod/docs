# Vietnam Data API Reference

API endpoints for accessing Vietnam-specific data.

## Endpoints

### GSO Indicators

```http
GET /api/v1/indicators?source=GSO
```

Parameters:
- `indicator` - GSO indicator code
- `start_year` - Start year
- `end_year` - End year
- `frequency` - monthly, quarterly, annual

### Customs Trade Data

```http
GET /api/v1/trade/vietnam
```

Parameters:
- `flow` - exports, imports, balance
- `partner` - country code
- `hs_code` - HS commodity code
- `period` - YYYY-MM format

### Stock Market Data

```http
GET /api/v1/stocks/vietnam
```

Parameters:
- `symbol` - Stock symbol (e.g., VCB)
- `index` - VNINDEX, HNX, UPCOM
- `start_date` - YYYY-MM-DD
- `end_date` - YYYY-MM-DD

## Response Format

```json
{
  "success": true,
  "data": [...],
  "metadata": {
    "source": "GSO",
    "last_updated": "2024-03-15",
    "total_records": 100
  }
}
```

## Error Codes

| Code | Message |
|------|---------|
| 400 | Invalid parameters |
| 401 | Authentication required |
| 404 | Data not found |
| 429 | Rate limit exceeded |
