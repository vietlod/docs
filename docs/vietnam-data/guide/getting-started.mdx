# Getting Started with Vietnam Data

This guide will help you quickly access Vietnam economic data through ECODATA.

## Quick Start

<Steps>

1. **Create an account** at [ECODATA](https://ecodata.khoviet.com)
2. **Navigate to Vietnam Data** section
3. **Browse data sources** (GSO, Customs, Stocks, etc.)
4. **Select indicators** you need
5. **Export data** in your preferred format

</Steps>

## Available Data Sources

| Source | Type | Best For |
|--------|------|----------|
| GSO | Official statistics | Macro analysis |
| Customs | Trade data | Trade research |
| Stock Market | Financial data | Investment analysis |
| Surveys | Microdata | Academic research |
| Indices | Rankings | Policy analysis |

## Data Formats

ECODATA supports multiple export formats:

| Format | Use Case |
|--------|----------|
| CSV | Spreadsheet analysis |
| Excel | Reports, charts |
| JSON | API integration |
| Stata | Statistical analysis |

## API Access

For programmatic access, see our [API Reference →](./api-reference)

```python
import requests

# Example: Get Vietnam GDP data
response = requests.get(
    "https://ecodata.khoviet.com/api/v1/indicators",
    params={
        "source": "GSO",
        "indicator": "GSO_GDP_Q_GR",
        "start_year": 2020
    },
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)
```
