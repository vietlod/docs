# Vietnam Industrial Production Index (IIP)

The **Index of Industrial Production (IIP)** measures monthly changes in industrial output.

## Key Indicators

| Code | Indicator | Frequency |
|------|-----------|-----------|
| GSO_IIP_M | IIP index (2015=100) | Monthly |
| GSO_IIP_M_GR | IIP growth (y-o-y) | Monthly |
| GSO_IIP_MANUF | Manufacturing IIP | Monthly |
| GSO_IIP_MINING | Mining IIP | Monthly |
