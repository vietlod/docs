# VN-Index

**VN-Index** is Vietnam's most widely followed stock market index, tracking all companies listed on HOSE.

## Index Information

| Metric | Value |
|--------|-------|
| Exchange | HOSE |
| Launch date | July 28, 2000 |
| Base value | 100 |
| Components | All HOSE stocks |
| Weighting | Free-float market cap |
