# WTO Trade Statistics

The **World Trade Organization (WTO)** provides comprehensive statistics on international trade in goods and services.

## Available Data

| Database | Description | Coverage |
|----------|-------------|----------|
| **Merchandise Trade** | Goods trade data | 164 members |
| **Commercial Services** | Services trade | 164 members |
| **Trade Profiles** | Country trade summaries | 164 members |
| **Tariff Profiles** | Applied tariff rates | 164 members |

## Key Features

- Official trade data
- WTO methodology
- Real-time updates
- Historical series

## Vietnam WTO Membership

Vietnam joined WTO in **January 2007** and is an active member.
