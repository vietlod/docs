# WTO Merchandise Trade

WTO provides detailed merchandise trade statistics by product and partner.

## Key Indicators

| Indicator | Description |
|-----------|-------------|
| Total exports | Merchandise exports value |
| Total imports | Merchandise imports value |
| Trade balance | Exports minus imports |
| Trade shares | % of world trade |
