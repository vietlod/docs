# ADB Key Indicators

The **Key Indicators for Asia and the Pacific** is ADB's flagship statistical publication.

## Indicator Categories

| Category | Indicators |
|----------|------------|
| People | Population, labor, education, health |
| Economy | GDP, prices, trade, finance |
| Environment | Energy, emissions, resources |
| Governance | Institutions, transparency |

## Coverage

- 49 ADB member countries
- Time series from 1965
- Annual updates
