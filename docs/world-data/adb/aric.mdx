# Asia Regional Integration Center (ARIC)

**ARIC** tracks regional economic integration and cooperation in Asia.

## Key Databases

| Database | Description |
|----------|-------------|
| Integration Indicators | Trade, investment, money |
| FTA Database | Free trade agreements |
| Economic Outlook | Regional forecasts |

## Integration Indicators

- Intraregional trade share
- FDI integration
- Financial integration
- Exchange rate co-movement

## Vietnam Integration

| Indicator | 2015 | 2023 |
|-----------|------|------|
| Intra-ASEAN trade | 22% | 25% |
| Intra-ASEAN FDI | 18% | 21% |
