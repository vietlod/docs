# Asian Development Bank (ADB) Data

The **Asian Development Bank** provides comprehensive economic data for Asia and Pacific region.

## Available Databases

| Database | Description | Coverage |
|----------|-------------|----------|
| **Key Indicators** | Core economic indicators | 49 members |
| **ARIC** | Regional integration indicators | ASEAN+3 |
| **SDG Dashboard** | SDG progress in Asia | 49 members |

## Key Features

- Asia-Pacific focus
- Regional comparisons
- Integration indicators
- Economic forecasts

## Vietnam in ADB Data

Vietnam is a founding member of ADB (1966) with comprehensive data coverage.
