# Worldwide Governance Indicators (WGI)

**WGI** measures governance across six dimensions for 200+ countries.

## Six Dimensions

| Dimension | Description |
|-----------|-------------|
| Voice and Accountability | Political participation |
| Political Stability | Violence, terrorism risk |
| Government Effectiveness | Public service quality |
| Regulatory Quality | Business regulations |
| Rule of Law | Contract enforcement |
| Control of Corruption | Corruption perception |
