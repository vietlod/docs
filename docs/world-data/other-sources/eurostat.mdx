# Eurostat

**Eurostat** is the statistical office of the European Union.

## Key Features

- 27 EU member states
- High-quality harmonized data
- Regular updates
- Comprehensive coverage

## Relevance for Vietnam

Eurostat provides:
- EU-Vietnam trade data
- EU FDI in Vietnam
- EU import statistics
