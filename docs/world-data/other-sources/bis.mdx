# Bank for International Settlements (BIS)

**BIS** provides financial and monetary statistics for central banks and researchers.

## Key Databases

| Database | Description |
|----------|-------------|
| Credit statistics | Credit to non-financial sector |
| Debt securities | International debt issuance |
| Derivatives | OTC derivatives market |
| Banking statistics | Cross-border banking flows |
| Exchange rates | Effective exchange rates |

## Key Indicators

- Credit-to-GDP ratio
- Debt service ratio
- Real effective exchange rate
- Cross-border claims
