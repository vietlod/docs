# Federal Reserve Economic Data (FRED)

**FRED** provides 800,000+ time series from 100+ sources.

## Key Categories

| Category | Series |
|----------|--------|
| Interest rates | Fed funds, Treasury yields |
| Exchange rates | USD pairs |
| Prices | CPI, PPI, PCE |
| Labor | Employment, wages |
| Production | Industrial output |

## Global Coverage

FRED includes international data:
- World Bank indicators
- OECD data
- IMF statistics

## API Access

```python
# ECODATA proxies FRED for convenience
params = {
    "source": "FRED",
    "series": "FEDFUNDS",
    "start_date": "2020-01-01"
}
```
