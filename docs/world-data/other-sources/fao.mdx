# Food and Agriculture Organization (FAO)

**FAOSTAT** provides food and agriculture statistics for 245 countries.

## Key Databases

| Database | Description |
|----------|-------------|
| Production | Crops, livestock, fisheries |
| Trade | Food imports/exports |
| Food security | Hunger, nutrition |
| Land use | Agricultural land |
| Emissions | Agricultural emissions |
