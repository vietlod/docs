# ILO Employment Statistics

ILOSTAT provides detailed employment and labor force data.

## Key Indicators

| Indicator | Description |
|-----------|-------------|
| EMP | Employment, total |
| UNE | Unemployment, total |
| LFP | Labor force participation rate |
| EAP | Economically active population |
