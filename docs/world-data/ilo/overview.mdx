# ILO Labor Statistics

The **International Labour Organization (ILO)** provides comprehensive labor market data through ILOSTAT.

## Key Databases

| Database | Description | Coverage |
|----------|-------------|----------|
| **ILOSTAT** | Core labor statistics | 180+ countries |
| **Wages** | Wage and earnings data | 100+ countries |
| **Decent Work** | Working conditions | 180+ countries |

## Indicator Categories

- Employment and unemployment
- Wages and earnings
- Working time
- Labor force participation
- Informal employment
- Child labor
