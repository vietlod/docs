# ILO Wage Statistics

ILOSTAT provides wage and earnings data across economies.

## Key Indicators

| Indicator | Description |
|-----------|-------------|
| Average wages | Mean monthly earnings |
| Minimum wages | Statutory minimum wage |
| Wage growth | Real wage index |
| Gender wage gap | Male-female earnings ratio |

## Wage Data Coverage

- 100+ countries
- Annual updates
- By sector and occupation

:::info Currency
Wages typically reported in local currency and USD.
