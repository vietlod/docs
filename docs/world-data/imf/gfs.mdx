# Government Finance Statistics (GFS)

The **Government Finance Statistics (GFS)** database provides fiscal data for general government and its subsectors.

## Key Indicators

### Revenue

| Code | Indicator |
|------|-----------|
| G1 | Total revenue |
| G11 | Taxes |
| G12 | Social contributions |
| G13 | Grants |
| G14 | Other revenue |

### Expenditure

| Code | Indicator |
|------|-----------|
| G2 | Total expenditure |
| G21 | Compensation of employees |
| G22 | Use of goods and services |
| G24 | Interest |
| G25 | Subsidies |
| G27 | Social benefits |

### Balance and Debt

| Code | Indicator |
|------|-----------|
| GGNLB | Net lending/borrowing |
| GGD | Gross debt |
| GGND | Net debt |
