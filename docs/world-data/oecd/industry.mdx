# OECD Industry Statistics

OECD provides comprehensive industry and structural business statistics.

## Key Databases

| Database | Description |
|----------|-------------|
| STAN | Structural Analysis Database |
| ANBERD | Business R&D |
| STRI | Services Trade Restrictiveness |

## STAN Database

- Value added by industry
- Employment by industry
- Investment by industry
- ISIC Rev 4 classification

---

:::info Coverage
Covers manufacturing, services, and technology sectors.
:::
