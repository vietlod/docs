# OECD Data Overview

The **Organisation for Economic Co-operation and Development (OECD)** provides high-quality economic data for 38 member countries and key partners.

## Available Databases

| Database | Description | Coverage |
|----------|-------------|----------|
| **National Accounts** | GDP, expenditure, production | 38+ countries |
| **Industry Statistics** | Manufacturing, services | 38+ countries |
| **Trade Statistics** | Goods and services trade | 38+ countries |
| **Main Economic Indicators** | Leading indicators | 38+ countries |

## Member Countries

OECD includes major economies:
- **North America**: USA, Canada, Mexico
- **Europe**: Germany, France, UK, Italy, Spain, etc.
- **Asia-Pacific**: Japan, Korea, Australia, New Zealand

## Key Partner Countries

Vietnam is an **OECD Key Partner**, with data available in many databases.
