# OECD Trade Statistics

OECD provides detailed trade statistics and analytical databases.

## Key Databases

### Trade in Value Added (TiVA)
- Domestic value added in exports
- Foreign value added content
- Global value chain participation

### Bilateral Trade Statistics
- Trade by partner
- Trade by commodity

## TiVA Indicators

| Indicator | Description |
|-----------|-------------|
| DVAR | Domestic value added in exports |
| FVAR | Foreign value added in exports |
| GVC_BWD | Backward GVC participation |
| GVC_FWD | Forward GVC participation |

---

:::tip GVC Analysis
TiVA is essential for understanding global value chain dynamics.
:::
