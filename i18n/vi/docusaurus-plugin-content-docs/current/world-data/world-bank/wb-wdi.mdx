# World Development Indicators (WDI) Guide

**World Development Indicators (WDI)** là cơ sở dữ liệu hàng đầu của World Bank về chỉ số phát triển, chứa hơn 1.600 chỉ số chuỗi thời gian cho 217 nền kinh tế.

## Cách truy vấn dữ liệu WDI

<Steps>

1. **Điều hướng đến phần World Bank** trong ECODATA
2. **Chọn nhóm chỉ số** (ví dụ: Kinh tế và Tăng trưởng)
3. **Chọn chỉ số cụ thể** từ danh sách
4. **Lọc theo quốc gia** - chọn quốc gia riêng lẻ hoặc khu vực
5. **Đặt khoảng thời gian** - có sẵn từ 1960 đến hiện tại
6. **Xuất dữ liệu** theo định dạng bạn muốn

</Steps>

## Ví dụ sử dụng trong nghiên cứu kinh tế lượng

### Phân tích tăng trưởng kinh tế

Sử dụng các chỉ số GDP để phân tích:
- Tăng trưởng GDP theo thời gian
- So sánh giữa các quốc gia
- Phân tích cấu trúc kinh tế (nông nghiệp, công nghiệp, dịch vụ)

### Nghiên cứu thương mại quốc tế

Sử dụng các chỉ số thương mại để:
- Phân tích xuất/nhập khẩu
- Nghiên cứu cán cân thanh toán
- Phân tích dòng vốn FDI

### Phân tích xã hội và phát triển

Sử dụng các chỉ số xã hội để:
- Nghiên cứu nghèo đói và bất bình đẳng
- Phân tích giáo dục và y tế
- Nghiên cứu dân số và lao động

## Ví dụ API

### Ví dụ Python

```python
import requests

# Truy vấn nhiều chỉ số WDI cho Việt Nam
url = "https://ecodata.khoviet.com/api/v1/indicators/batch"

payload = {
    "source": "WB",
    "indicators": [
        "NY.GDP.MKTP.CD",
        "NY.GDP.MKTP.KD.ZG",
        "FP.CPI.TOTL.ZG"
    ],
    "countries": ["VN", "TH", "ID", "MY", "PH"],
    "start_year": 2015,
    "end_year": 2023
}

response = requests.post(url, json=payload)
data = response.json()

# In kết quả
for item in data['data']:
    print(f"{item['country']}: {item['indicator']} = {item['value']} ({item['year']})")
```

### Định dạng đầu ra

