# Stock Hub Indicators

Stock Hub supports stock market and listed-firm analysis. Data is organized by symbol, industry, index group, and financial statement tables.

## Data Groups

| Group | Content |
| --- | --- |
| Price | Prices, volume, price changes, and trading history. |
| Symbol profile | Exchange, ICB industry, index group, and instrument type. |
| Balance Sheet | Assets, liabilities, and equity. |
| Income Statement | Revenue, profit, expenses, and EPS. |
| Cash Flow | Operating, investing, and financing cash flow. |
| Notes | Financial statement notes. |
| Events Calendar | Dividends, meetings, issuance events, and corporate calendar. |

## Workflow

1. Open Stock Hub.
2. Search a stock symbol or filter by index group or industry.
3. Open the symbol detail page.
4. Choose Price, Balance Sheet, Income Statement, Cash Flow, Notes, or Events.
5. Select year or quarter when viewing financial statements.
6. Export tables or use the data in external analysis.

## Source Priority

The app prioritizes database and cache reads first, then falls back to service sources or offline files when needed. For financial statements, VCI offline parquet data is used for balance sheet, income statement, cash flow, and notes; VietStock can be a fallback or supplementary source.

## Research Notes

1. Price data is usually daily; financial statements are quarterly or annual.
2. Normalize industry and exchange codes for cross-sectional analysis.
3. When combining with macro data, choose a common frequency or aggregate appropriately.
4. Notes may require text processing or indicator mapping before modeling.
