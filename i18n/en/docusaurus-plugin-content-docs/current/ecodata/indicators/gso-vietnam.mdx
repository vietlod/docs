# GSO and Vietnam Indicators

GSO Explorer focuses on official Vietnam statistics. This module is suitable when users need indicators by province, year, sector group, or official statistical report.

## Main Datasets

| Dataset | Purpose |
| --- | --- |
| KTXH | Socio-economic indicators by reporting period. |
| NGTK-CN | National statistical yearbook data for national time series. |
| NGTK-TINH | Provincial statistical yearbook data for province-year panels. |
| VHLSS aggregate | Aggregated indicators from official VHLSS reports, not microdata. |

## GSO Indicator Groups

| Group | Examples |
| --- | --- |
| Population and labor | Population, population density, labor force, employment, unemployment. |
| National accounts | GRDP/GDP, sector structure, current prices, constant prices. |
| Industry | IIP, output, firms, manufacturing sectors. |
| Agriculture | Area, production, livestock, fisheries. |
| Trade and services | Retail, tourism, transport, local import-export. |
| Prices | CPI and price index groups. |
| Social indicators | Education, health, living standards, poverty, income. |

## GSO Explorer Workflow

1. Open GSO Explorer.
2. Select KTXH, NGTK-CN, NGTK-TINH, or VHLSS.
3. Use Browse Catalog to find an indicator group.
4. Select indicators.
5. Select provinces and years.
6. View province-year pivot tables and heatmap, line, bar, or area charts.
7. Export Excel or CSV with metadata when needed.

## When to Use GSO Instead of Global Data

Use GSO when the research focuses on Vietnam, requires official province-level statistics, or needs to match domestic yearbooks and reports. Use Global Data when comparing Vietnam with other countries.

## Data Quality Notes

1. Provincial indicator names may change by year or source report.
2. Administrative boundary changes can affect long time series.
3. Always check units and current or constant price basis before combining indicators.
4. For PDF or Excel extraction sources, preview several sample years before export.
