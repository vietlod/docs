# Macro Survey and VHLSS

This group contains two different data layers: province-level Macro Survey data and household or individual-level VHLSS Micro data. Users must keep the unit of analysis clear to avoid invalid merges.

## Macro Survey

Macro Survey collects provincial indices that often represent institutions, governance, and public service quality.

| Index | Meaning |
| --- | --- |
| PCI | Provincial competitiveness and business environment. |
| PAPI | Provincial governance and public administration. |
| ICT | Information and communication technology readiness and adoption. |
| PAR | Public administrative reform. |
| SIPAS | Citizen satisfaction with public administrative services. |

Workflow:

1. Open Survey Explorer.
2. Select a survey or indicator group.
3. Filter province and year.
4. Open indicator detail to read the definition.
5. Review table, chart, and export CSV when needed.

## VHLSS Aggregate

VHLSS aggregate is available in GSO Explorer. It includes indicators summarized from official reports. Use this group for living standards, income, expenditure, poverty, education, or health indicators by province or population group.

## VHLSS Micro and Variable Hub

VHLSS Micro uses Stata microdata files. Variable Hub helps users select wave, dataset, and variables before creating an analysis dataset.

| Step | Action |
| --- | --- |
| 1 | Select waves, for example 2002-2024 depending on availability. |
| 2 | Select datasets or data table groups. |
| 3 | Filter variables by keyword, label, dataset, or wave. |
| 4 | Preview sample data to validate variables. |
| 5 | Review descriptive summary. |
| 6 | Export as single-wave merge, cross-wave panel, or pooled append. |

## VHLSS Micro Export Modes

| Mode | When to use |
| --- | --- |
| Single-wave merge | A one-year survey study that needs several tables merged within one wave. |
| Cross-wave panel | A time-based study with appropriate panel keys. |
| Pooled append | Combining waves to increase observations without tracking the same unit over time. |

## Method Notes

1. Macro Survey is province-level; VHLSS Micro is household or individual-level.
2. VHLSS Micro requires careful checks on weights, province codes, waves, household or person keys, and variable name changes.
3. Full export can depend on account permissions; preview first before large exports.
