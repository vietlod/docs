# Data Source Groups

Ecodata brings many economic data sources into a unified workflow. When choosing a source, start from the unit of analysis rather than the organization name.

## Source Map by Unit of Analysis

| Unit of analysis | Suitable sources | Module |
| --- | --- | --- |
| Country | World Bank, IMF, OECD, Eurostat, ILO, FAO, WTO, UNCTAD, ADB, FRED, OWID | Dashboard |
| Vietnam province | GSO KTXH, NGTK-TINH, VHLSS aggregate, Macro Survey | GSO, Survey |
| Commodity and trade partner | Vietnam Customs | Customs |
| Household or individual | VHLSS Micro | VHLSS Variable Hub |
| Listed firm | Stock Hub, financial statements, Events Calendar | Stocks |
| Provincial governance index | PCI, PAPI, PAR, SIPAS, ICT | Macro Survey |

## Global Data

This group is suitable for cross-country comparison, national panel data, macroeconomic testing, and standard control variables. Sources supported by the application and technical documentation include World Bank, IMF, OECD, Eurostat, ILO, FAO, OWID, FRED, WTO, UNCTAD, ADB, and SAMADB.

Common indicators include GDP growth, GDP per capita, inflation, exchange rate, interest rate, balance of payments, labor, wages, productivity, trade, population, education, health, governance, risk, and development indicators.

## Vietnam GSO Data

GSO has several data layers:

| Dataset | Content |
| --- | --- |
| KTXH | Socio-economic indicators by reporting period. |
| NGTK-CN | National statistical yearbook data. |
| NGTK-TINH | Provincial statistical yearbook data. |
| VHLSS aggregate | Aggregated indicators from official VHLSS reports. |

GSO data is suitable when the research needs official Vietnam indicators by province, year, or report group.

## Customs

Customs data comes from General Department of Customs reports. It is suitable for research on merchandise trade, export markets, import partners, provincial trade structure, and transport modes.

Main dimensions include commodity, partner country, province, transport mode, reporting month or quarter, and report status.

## Macro Survey and VHLSS

Macro Survey includes provincial indicators such as PCI, PAPI, ICT, PAR, and SIPAS. These are useful for business environment, public governance, administrative reform, and public service quality variables.

VHLSS Micro is different from VHLSS aggregate. It uses Stata microdata by wave, dataset, and variable. Users select waves, tables, variables, preview records, and choose merge or append mode.

## Stock Hub

Stock Hub is for listed firms, capital market research, and financial data. The app reads from database and cache first, then falls back to service sources or offline files when needed.

Data groups include prices, volumes, market cap, symbol profile, exchange, ICB industry, index groups, balance sheet, income statement, cash flow, notes, and corporate events.

## Practical Source Selection

1. For cross-country research, start with Global Data.
2. For Vietnam province-level research, start with GSO or Macro Survey.
3. For import-export questions, start with Customs.
4. For household or individual analysis, start with VHLSS Micro.
5. For listed firms, start with Stock Hub.
6. If unsure, use AI Chat to turn the research question into candidate sources and indicators.
